# 🚀 Proyecto API REST -

Este repositorio contiene el desarrollo de una API RESTful segura y documentada, elaborada como parte de las asignaciones para la clase impartida por el **Profesor Santos Nehemias**. 

El proyecto demuestra la implementación de buenas prácticas de desarrollo backend, incluyendo autenticación, manejo de base de datos y documentación automatizada.

---

## 🛠️ Tecnologías y Herramientas

* **Arquitectura:** API REST
* **Autenticación:** JWT (JSON Web Tokens)
* **Documentación:** Swagger (OpenAPI)
* **Base de Datos:** SQL (Estructura y datos semilla incluidos)

---

## 📂 Estructura Principal del Proyecto

El repositorio cuenta con archivos clave para la configuración rápida del entorno y la base de datos:

* `.env.example`: Plantilla de las variables de entorno necesarias para ejecutar el proyecto (credenciales, secretos de JWT, etc.).
* `database/schema.sql`: Script principal que contiene la estructura de la base de datos (tablas, relaciones, constraints).
* `database/seed...`: Archivo(s) de poblado de datos (seeds) para insertar registros iniciales y facilitar las pruebas de la API.
